# Border Radius

| Token | Value | Default usage |
|---|---|---|
| base | 24px | Cards, modals, sections, dropdown menus, popovers, tables, accordion, sidebars |
| default | 999px | Buttons, inputs, badges, tooltips, tabs, pagination items |
| sm | 4px | Checkboxes, tiny elements |
| full | 9999px | Avatars, toggles, dot indicators |

## Rules

- 24px (base) is the default radius for container-type elements (cards, modals, panels)
- 999px (default) is the default radius for interactive controls (buttons, inputs, badges) giving them a pill shape
- Never use arbitrary radius values outside this scale
- Radius must be consistent within each component family
