# Pricing Cards with Expandable Comparison

Create a pricing section as one vertically stacked block inside a max-width container. At the top, place a centered heading on its own line; directly below the heading, a centered supporting paragraph at a comfortable reading width.

Below the heading block, place a horizontal row of three pricing cards of equal width with consistent gutters between them. Inside each card, stack from top to bottom: a plan name on its own line; a short single-line description directly below the name; a large dominant price on a line directly below the description; one full-width primary call-to-action button; and a short vertical list of headline feature rows (each row composed of a small circular check icon at the left and a short feature label to its right on the same baseline).

Centered directly below the card row on its own line, place a single text link (e.g. "Compare all features") that acts as a disclosure toggle. When the link is activated, reveal a wide feature comparison table directly beneath it. The first row of the table repeats the plan names as column headers (one per plan); the table body rows each pair a feature label in the left-most column with a small positive check icon, a small neutral dash icon, or a short text value in each plan column — all vertically centered within the row.

On narrow viewports the three cards stack vertically in their original left-to-right order, the comparison link remains centered below the cards, and the revealed table collapses into one stacked card per plan that repeats the feature list with the same icons; cards stack vertically in the original column order.
