# Stacked Pricing Rows

Create a pricing section as one vertically stacked block inside a max-width container. At the top, place a centered heading on its own line; directly below the heading, a centered supporting paragraph at a comfortable reading width.

Below the heading block, place a vertical stack of three plan rows separated by thin horizontal dividers spanning the full container width. Each row is a single horizontal band containing three columns, from left to right:

**Left column** — a plan name on its own line with a short single-line description directly below the name.

**Middle column** — a vertical list of short feature labels stacked one per line (no leading icons), vertically centered within the row.

**Right column** — a large dominant price on its own line with a small call-to-action button directly below the price, both right-aligned within the column.

On narrow viewports each row collapses into a stacked card containing the plan name, description, feature labels, price, and call-to-action button in a single vertical reading order with the price and button at the bottom; cards stack vertically with thin dividers between them.
