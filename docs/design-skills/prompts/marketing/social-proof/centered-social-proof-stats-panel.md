# Centered Social Proof Stats Panel

Create a social proof section inside a centered page container with comfortable side padding. At the top, place a centered header block composed of a large heading centered on its own line followed by a centered supporting paragraph directly beneath it at a moderate reading width. Below the paragraph, place a horizontal row of two text links sitting side by side on the same baseline, centered as a pair; each link label includes a trailing arrow icon.

Below the header, place one wide rounded container spanning the full container width. Inside the container, arrange four stat blocks in a single horizontal row on the same baseline, separated by thin vertical dividers between adjacent blocks. Each stat block is a left-aligned vertical stack composed of one large prominent numeric value on its own line at the top and a short descriptive label directly beneath the value.

On narrow viewports the two text links stack vertically, and the four stat blocks collapse into a two-column grid with the vertical dividers removed; preserve left-to-right top-to-bottom reading order throughout.
