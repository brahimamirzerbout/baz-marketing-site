# Rule-Aligned Social Proof Stats

Create a social proof section inside a centered page container with comfortable side padding. At the top, place a large heading centered on its own line.

Below the heading, place a single thin horizontal rule spanning the full container width. Along this rule, position four equal-width stat columns on the same baseline with consistent spacing between them. Each column is a centered vertical stack containing, in order: a small pill-shaped badge on its own line sitting directly above the horizontal rule (composed of a tiny leading icon and a short category label, both inside a rounded container); one large prominent numeric value on its own line directly below the rule; a short descriptive label directly beneath the value. Keep all content within each column horizontally centered so the badges, numbers, and labels align vertically.

On narrow viewports collapse the four columns into a two-column grid first, then into a single column on the narrowest breakpoints; replace the horizontal rule with consistent vertical spacing between items and preserve top-to-bottom reading order.
