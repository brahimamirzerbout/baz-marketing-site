# Centered Social Proof Stats Row

Create a social proof section inside a centered page container with comfortable side padding. Place a single horizontal row of three stat blocks centered on the page on the same baseline with generous spacing between them. Each stat block is a centered vertical stack composed of one large prominent numeric value (e.g. "73M+") on its own line at the top and a short descriptive label directly beneath the value. Keep all text within each block horizontally centered.

On narrow viewports collapse the three stat blocks into a single column, stacking them vertically in left-to-right reading order with consistent spacing between them.
