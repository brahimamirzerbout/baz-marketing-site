# Split Social Proof Intro With Stat List

Create a social proof section split into two equal-width columns on large 1280px width breakpoints that spans the full available width of the page container.

The **left column** is a left-aligned vertical stack with comfortable internal padding containing, in order: a large left-aligned multi-line heading on its own line; a supporting paragraph directly beneath the heading at a comfortable reading width; a social-proof row composed of a tight horizontal cluster of slightly overlapping small circular avatar thumbnails on the left and a single line of caption copy to their right that includes a numerically emphasized segment (e.g. a user count rendered in bold); and a compact outlined button on its own line below the social-proof row whose label includes a trailing arrow icon.

The **right column** uses a contrasting filled background surface that spans the full column area. Allow a very subtle geometric wave or line texture behind the column content so it reads as a soft decorative surface without competing with the text. Inside the column, place a vertical list of three to four stat entries with consistent spacing between them. Each entry is a horizontal row composed of a small rounded-square icon container anchored to the left (holding a single centered glyph) and a text block to the right of the icon containing a bold stat headline on its own line (combining a prominent numeric value with a short label) and a short multi-line description paragraph directly beneath the headline.

On narrow viewports stack the two columns vertically with the intro column first and the stat-list column second; each column stretches to the full content width and preserves all internal stacks in their original reading order.
