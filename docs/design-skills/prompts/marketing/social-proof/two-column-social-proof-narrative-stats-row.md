# Two-Column Social Proof Narrative With Stats Row

Create a social proof section inside a centered page container with comfortable side padding. At the top, place a two-column layout on large 1280px width breakpoints.

The **left column** is a left-aligned vertical stack containing, in order: a large left-aligned multi-line heading on its own line; a supporting paragraph directly beneath the heading at a comfortable reading width.

The **right column** is a left-aligned vertical stack containing multiple body paragraphs stacked vertically with consistent spacing between them, providing extended narrative or descriptive copy.

Below the two-column block, after generous vertical space, place a single horizontal row of four stat blocks spanning the full container width on the same baseline with consistent spacing between them. Each stat block is a centered vertical stack composed of one large prominent numeric value on its own line at the top and a short descriptive label directly beneath the value.

On narrow viewports stack the two columns vertically in reading order with the heading column first and the narrative column second; the stat row collapses into a two-column grid preserving left-to-right top-to-bottom reading order.
