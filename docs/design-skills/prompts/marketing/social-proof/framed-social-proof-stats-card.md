# Framed Social Proof Stats Card

Create a social proof section inside a centered page container with comfortable side padding. Place one large rounded outer container with a thin outer boundary that frames the entire section content so it reads as a single card.

Inside the container, place a horizontal row of four equal-width stat tiles on the same baseline with consistent gutters between them. Each tile is its own rounded rectangular panel containing a centered vertical stack composed of, in order: a small icon centered at the top; one large prominent numeric value on its own line directly below the icon; a short descriptive label directly beneath the value. Keep all content within each tile horizontally centered.

Below the stat tiles, after comfortable vertical space, place a left-aligned trust row composed of a short introductory label (e.g. "Trusted by:") followed by a horizontal row of brand logos, each rendered as a small monochrome glyph with consistent spacing between them, all sitting on the same baseline as the label.

On narrow viewports collapse the four tiles into a two-column grid first, then into a single column on the narrowest breakpoints; the trust row wraps naturally beneath the label.
