# Social Proof Carousel With Stats Row

Create a social proof section inside a centered page container with comfortable side padding. At the top, place a centered header block composed of a large heading centered on its own line followed by a single centered text link on its own line directly beneath the heading whose label includes a trailing arrow icon.

Below the header, place a horizontal carousel region spanning the full container width. The carousel displays two large rounded rectangular image placeholders side by side in the visible frame with consistent gutters between them. Position a circular previous-arrow navigation control on the left edge of the carousel and a circular next-arrow navigation control on the right edge, both vertically centered relative to the image placeholders.

Below the carousel, place a single horizontal row of six stat blocks on the same baseline with consistent spacing between them. Each stat block is a centered vertical stack composed of one large prominent numeric value on its own line at the top and a short descriptive label directly beneath the value. Keep all content within each block horizontally centered.

On narrow viewports the carousel shows one image placeholder at a time, the navigation controls remain anchored to the left and right edges, and the stat row collapses into a two-column or three-column wrapped grid preserving left-to-right top-to-bottom reading order.
