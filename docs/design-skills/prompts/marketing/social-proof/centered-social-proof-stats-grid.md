# Centered Social Proof Stats Grid

Create a social proof section inside a centered page container with comfortable side padding. At the top, place a centered header block composed of a large heading centered on its own line followed by a centered supporting paragraph directly beneath it at a moderate reading width. Below the paragraph, place a horizontal row of two text links sitting side by side on the same baseline, centered as a pair; each link label includes a trailing arrow icon.

Below the header, place a four-column by two-row grid of eight stat blocks with consistent gutters between columns and rows. Each block is a centered vertical stack containing, in order: a small icon centered at the top; one large prominent numeric value on its own line directly below the icon; a short multi-line descriptive label directly beneath the value. Keep all content within each block horizontally centered.

On narrow viewports reduce the column count stepwise (four columns, then two, then one) until the blocks form a single column on the narrowest breakpoints; preserve left-to-right top-to-bottom reading order throughout.
