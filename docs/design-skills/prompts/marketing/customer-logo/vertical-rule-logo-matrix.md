# Vertical-Rule Logo Matrix

Create a customer logos section as one rectilinear grid inside a page-width container with comfortable side padding, arranged on large 1280px width breakpoints with exactly four columns and three rows for twelve partner entries in reading order left-to-right then top-to-bottom. Draw thin vertical dividing rules between whole columns so the grid reads as four lanes separated by hairlines; do not run horizontal rules between rows, relying instead on consistent vertical spacing and uniform row padding to separate the three tiers visually.

Within **each cell**, treat the partner identity as a single horizontal unit: a compact brand mark or pictogram on the leading side and the company wordmark or name on the trailing side, aligned on the same baseline as one pair. Center that icon-and-wordmark pair as a whole inside the cell both horizontally and vertically. Keep column widths equal and cell padding uniform so the matrix stays balanced across all twelve entries.

On narrow viewports reflow into fewer columns (for example two columns or a single column) while keeping the icon-plus-wordmark pairing intact per cell. Adjust the vertical rules so they only separate adjacent columns of cells that appear on the same row, not individual stacked rows.
