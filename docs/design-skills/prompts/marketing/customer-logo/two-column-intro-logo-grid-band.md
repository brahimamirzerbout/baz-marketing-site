# Two-Column Intro Logo Grid Band

Create a customer logos section as one vertical stack inside a page-width container with comfortable side padding.

The **upper region** uses a two-column layout on large 1280px width breakpoints. The left column is a left-aligned vertical stack containing, in order: a prominent primary heading on its own line; and one supporting paragraph directly beneath the heading at a comfortable reading width. The right column holds a longer body block made of several successive paragraphs (for example three) that expand on scale, audiences, or proof points, all reading as one narrative column whose top edge aligns consistently with the left column's text block.

Leave clear vertical space beneath the two-column intro, then span a **full-width horizontal band** whose height fits one row of partner marks plus comfortable vertical padding above and below. Differentiate the inside surface of the band from the surrounding page by carrying a subtle repeating rectilinear grid motif behind the marks so the strip reads as a graph-paper-like field rather than an undifferentiated flat surface. Center a single row of exactly five partner marks inside the band with even horizontal spacing and consistent vertical alignment so each mark reads at similar scale.

On narrow viewports stack the intro columns with the heading-and-summary column first and the multi-paragraph column second; allow the marks inside the band to wrap into additional centered rows while keeping spacing even between items and consistent vertical gaps between wrapped lines.
