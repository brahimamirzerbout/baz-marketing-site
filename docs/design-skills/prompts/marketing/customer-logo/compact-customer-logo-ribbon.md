# Compact Customer Logo Ribbon

Create a customer logos section that reads as one shallow horizontal band spanning the full content width with comfortable vertical padding above and below the marks. Inside the band, center a single horizontal row of partner marks with uniform spacing between each mark; do not place any heading or supporting copy inside the band itself (any heading may sit above the band as a sibling block in the parent layout if needed). Set the band height to fit the tallest mark plus comfortable vertical padding so the strip reads as a compact ribbon rather than a tall hero region. On narrow viewports the marks wrap into additional centered rows inside the same band while maintaining even horizontal spacing and consistent vertical gaps between wrapped lines.
