# Staggered Integration Logo Lanes

Create a customer logos section split into two columns on large 1280px width breakpoints inside a centered page container with comfortable side padding.

The **left column** is a left-aligned vertical stack containing, in order from top to bottom: a multi-line primary heading on its own line that may include an inline segment naming the product or integration surface being promoted; one supporting paragraph directly beneath the heading describing automating or connecting existing tools; and a single full-width primary call-to-action control whose label invites exploration of integrations (the label text ends with a trailing arrow or chevron character after the wording).

The **right column** presents a field of partner marks organized as several parallel vertical lanes of tall portrait-oriented cells. Each lane stacks multiple cells one above another, and each cell holds exactly one centered partner mark. Offset neighboring lanes vertically so cells in adjacent columns do not share the same horizontal baselines, producing a staggered brick rhythm instead of a flush rectangular grid. Allow the composition to clip or fade at the top and along one outer side so the field implies more marks continue beyond the visible frame without listing every partner in view.

On narrow viewports stack the columns with the heading, paragraph, and CTA first and the staggered logo field second; reduce the lane count and simplify vertical offsets while preserving one mark per cell and maintaining the staggered rhythm where space allows.
