# Looping Customer Logo Track

Create a customer logos section where the partner marks sit inside a single overflowing horizontal track that spans the full content width. Duplicate the full sequence of marks end to end so the track can scroll or animate horizontally as one continuously looped strip. Apply a soft fade or gradient mask at the left and right edges of the visible area so the strip hints that more marks continue off-screen in both directions.

Set the track height to be shallow (the tallest mark plus comfortable vertical padding) so the section reads as a compact ribbon rather than a tall hero region. Provide a non-scrolling fallback when reduced-motion preferences are active by showing the marks in a centered static row that wraps into additional lines on narrow viewports while maintaining even horizontal spacing between items.
