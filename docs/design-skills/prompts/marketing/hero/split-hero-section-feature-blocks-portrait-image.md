# Split Hero Section with Feature Blocks and Portrait Image

Create a 100vh height hero split into two columns on large 1280px width breakpoints with a 64px horizontal gap between them inside a centered page container.

**Left column** (wider) is a left-aligned vertical stack containing, in order: a heading on its own line; a supporting paragraph directly below the heading at a comfortable reading width; a horizontal divider spanning the full width of the column on its own line directly below the paragraph. Below the divider, place two side-by-side feature blocks separated by a thin vertical divider on the same baseline. Each feature block is a vertical stack composed of a small leading icon at the top, a bold title line on its own line directly below the icon, a short description paragraph directly below the title, and one full-width CTA button at the bottom of the block (the left block uses a primary button and the right block uses a secondary button).

**Right column** holds one tall portrait-oriented photograph that fills the full height of the column alongside the left content (the image covers the column area edge-to-edge).

On narrow viewports stack the two columns vertically with the text column first and the image directly below it; the two feature blocks stack vertically in reading order and the thin vertical divider between them is replaced by a horizontal divider or removed entirely.
