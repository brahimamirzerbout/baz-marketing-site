# Centered Hero Section with Badge, CTAs, and Featured Image

Create a hero as one centered vertical stack on a full-width section background. Stack from top to bottom: a small pill-shaped badge composed of a tiny leading icon and a short label, both centered horizontally on its own line; a large centered heading on its own line directly below the badge; a centered supporting paragraph directly below the heading at a comfortable reading width; a horizontal row of two CTA buttons sitting side by side on the same baseline (a primary button on the left and a secondary button on the right), centered as a pair beneath the paragraph; finally, a single large centered image placed directly below the button row that spans most of the width of the container. On narrow viewports keep the same single-column centered stack, let the two CTA buttons wrap into a vertical pair if needed, and allow the image to span the full content width with comfortable side padding.
