# Hero with Stadium Showcase and Feature Row

Create a hero as a vertical stack of four distinct bands within a bounded max width so the section reads as one continuous page slice.

**Top bar** is a horizontal navigation row composed of a logo on the left, a short row of inline text nav links centered in the bar on the same baseline, and one compact rounded primary action button on the right.

**Intro split** sits directly below the top bar as a two-column layout on large 1280px width breakpoints. The wider left column carries only a very large, left-aligned, multi-line headline on its own line. The narrower right column carries a short three- to four-line supporting paragraph, left-aligned on its own line.

**Showcase image** spans the full width of the bounded container directly below the intro split as a single large image.

**Feature row** sits directly below the showcase image as a two-column layout separated by a thin vertical divider. Each column is a vertical stack composed of a small rounded-square icon container on its own line at the top, a medium heading on its own line directly below the icon, and a short paragraph directly below the heading at a comfortable reading width.

On narrow viewports stack the intro columns vertically with the headline first and the paragraph directly below it; let the showcase image span the full width of the slice; stack the bottom feature columns in reading order with the thin vertical divider becoming a horizontal rule between them.
