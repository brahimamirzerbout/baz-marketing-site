# Three-Tile Image Hero

Create a hero laid out as a small bento-style grid with clear gutters between cells inside a centered page container.

**Top tile** spans the full width of the grid as a single row. Its area is filled by a photographic background image edge-to-edge within the tile. Inside that tile, anchor content to the top-left corner with comfortable padding: a large multi-line headline on its own line, then directly below it a highly rounded pill-shaped primary button composed of a small leading icon beside a short label.

**Bottom row** splits into two equal-width tiles sitting side by side. Each tile repeats the same pattern at a smaller scale: a photographic background image filling the tile edge-to-edge; a shorter headline anchored to the top-left corner on its own line with comfortable padding; then the same highly rounded pill-shaped button treatment directly below the headline (composed of a small leading icon beside a short label for parity with the main tile).

On narrow viewports stack the three tiles vertically in reading order: the large top tile first, then the left bottom tile, then the right bottom tile; each tile spans the full content width with comfortable side padding.
