# Full-Width Hero with Overlay and Metric Row

Create a full-width hero section fixed at 640px height whose background is a photographic image that covers the entire section edge-to-edge. Layer a single semi-transparent overlay over the image so all foreground content remains readable.

**Main content block** is left-aligned with comfortable inset padding, containing a vertical stack in order: a heading on its own line; a supporting paragraph directly below the heading at a comfortable reading width; a horizontal row of two CTA buttons sitting side by side on the same baseline directly below the paragraph (the first button is composed of a short label paired with a trailing arrow icon; the second button is composed of a small leading video-style icon paired with a short label).

**Bottom metric row** is anchored along the bottom edge of the section: a horizontal row of four metric items on the same baseline. Each metric item is composed of one prominent number or short numeric value on the left and a two-line label immediately to the right of that value on the same horizontal axis (suitable for plan highlights, trust figures, or comparable copy).

On narrow viewports keep the main heading, paragraph, and buttons in a single left-aligned vertical stack, and wrap or stack the four metric items vertically in reading order.
