# Minimal Footer Social Bar

Create a footer section as a single horizontal bar inside a max-width container with comfortable side padding. On the left side of the bar, place one copyright line on its own baseline. On the right side of the same bar, place a horizontal row of small social-media icon links sitting side by side on the same baseline with consistent spacing between them.

On narrow viewports stack the copyright line above the social icon row, both centered, preserving reading order.
