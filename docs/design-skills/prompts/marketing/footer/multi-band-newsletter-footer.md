# Multi-Band Newsletter Footer

Create a footer section as one vertically stacked block inside a max-width container with comfortable side padding, composed of four distinct bands separated by generous vertical space.

**Newsletter band** — a centered vertical stack containing a large centered multi-line heading on its own line; directly below the heading, a single inline form row composed of a text input with a leading email icon and placeholder text on the left and a compact primary submit button directly to its right on the same baseline, centered as a pair beneath the heading.

**Sitemap band** — a horizontal row of four equal-width link columns on large breakpoints with consistent gutters between them. Each column is a left-aligned vertical stack containing a bold category heading on its own line at the top followed by a vertical list of plain text links, one per line, directly beneath the heading.

**Contact band** — a horizontal row of three contact blocks on the same baseline with consistent spacing between them. Each block is a horizontal composition of a small icon anchored to the left (representing the contact channel — phone, email, or location) and a text group to the right of the icon containing a short descriptive label on its own line and a prominent contact value directly beneath the label.

**Bottom bar** — a horizontal row spanning the full container width containing, from left to right: a brand mark composed of a small logo icon paired with the brand name to its right on the same baseline; a centered copyright line; and a compact language selector composed of a small flag icon paired with a language label and a trailing chevron, anchored to the right end of the bar.

On narrow viewports stack each band's columns vertically in reading order; the newsletter form input and button stack vertically; the bottom bar elements center and stack in order: brand mark, copyright, language selector.
