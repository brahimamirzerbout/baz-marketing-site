# Centered Brand Footer

Create a footer section as one centered vertical stack inside a max-width container with comfortable side padding. Stack from top to bottom: a brand mark composed of a small logo icon paired with the brand name to its right on the same baseline, centered horizontally on its own line; a single centered tagline paragraph directly below the brand mark at a moderate reading width; a horizontal row of navigation links sitting side by side on the same baseline, centered as a group directly below the tagline (each link is a short text label with consistent spacing between adjacent links); and a single centered copyright line on its own line at the bottom of the section, separated from the navigation row by comfortable vertical space.

On narrow viewports keep the entire stack centered; allow the navigation links to wrap into multiple lines if needed while preserving reading order.
