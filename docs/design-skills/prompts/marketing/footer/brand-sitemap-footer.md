# Brand Sitemap Footer

Create a footer section as one vertically stacked block inside a max-width container with comfortable side padding, composed of two distinct bands separated by generous vertical space.

**Main band** — split into two regions on large breakpoints. The left region is a left-aligned vertical stack containing a brand mark composed of a small logo icon paired with the brand name to its right on the same baseline on its own line; a short multi-line description paragraph directly below the brand mark at a moderate reading width; and a horizontal row of small social-media icon links sitting side by side on the same baseline with consistent spacing between them, placed directly below the description. The right region contains a horizontal row of four equal-width link columns with consistent gutters between them. Each column is a left-aligned vertical stack containing a bold category heading on its own line at the top followed by a vertical list of plain text links, one per line, directly beneath the heading.

**Copyright row** — a single centered copyright line on its own line spanning the full container width below the main band, separated by a thin horizontal rule.

On narrow viewports stack the main band regions vertically with the brand-and-social block first and the sitemap columns second; collapse the sitemap columns into a two-column grid or a single column in reading order; the copyright line remains centered at the bottom.
