# Primary Navigation Footer

Create a footer section as one vertically stacked block inside a max-width container with comfortable side padding, composed of two horizontal rows separated by generous vertical space.

**Primary row** — a single horizontal bar spanning the full container width containing, from left to right: a brand mark composed of a small logo icon paired with the brand name to its right on the same baseline; a centered horizontal row of navigation links sitting side by side on the same baseline with consistent spacing between them; and a horizontal row of small social-media icon links anchored to the right end of the bar on the same baseline with consistent spacing between them.

**Copyright row** — a single centered copyright line on its own line below the primary row.

On narrow viewports stack the primary row elements vertically in order: brand mark centered, navigation links centered (wrapping into multiple lines if needed), social icons centered; the copyright line remains centered at the bottom.
