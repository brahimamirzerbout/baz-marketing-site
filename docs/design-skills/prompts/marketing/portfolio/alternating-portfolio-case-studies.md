# Alternating Portfolio Case Studies

Create a portfolio section inside a centered page container with comfortable side padding. At the top, place a horizontal row composed of a left-aligned heading on the left and one compact outlined button (a short label paired with a trailing arrow icon) anchored to the right, both sitting on the same baseline. Directly below the heading, place a left-aligned supporting paragraph at a comfortable reading width.

Below the header, stack three case-study entries vertically with generous spacing between them. Each entry is a two-column split on large 1280px width breakpoints with consistent gutters. Alternate the column positions so the first entry places the image on the left and the content on the right, the second entry places the content on the left and the image on the right, and the third entry repeats the first arrangement.

Inside each **image column**, place one large image with rounded corners that fills the column area.

Inside each **content column**, stack a left-aligned vertical composition from top to bottom: a prominent project title on its own line; a text link on its own line directly below the title composed of a URL label paired with a small trailing external-link icon; a supporting paragraph directly below the link at a comfortable reading width; a horizontal row of small tech-stack or tool icons sitting side by side on the same baseline; and a text-style link at the bottom composed of a short label paired with a trailing arrow icon.

On narrow viewports, stack each entry as a single column with the image first and the content directly below it; preserve all internal reading order.
