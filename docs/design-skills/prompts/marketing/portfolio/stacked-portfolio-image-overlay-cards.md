# Stacked Portfolio Image Overlay Cards

Create a portfolio section inside a centered page container with comfortable side padding. At the top, place a centered heading on its own line; directly below the heading, a centered text link composed of a short label paired with a trailing arrow icon.

Below the header, stack three full-width cards vertically with consistent spacing between them. Each card is a large rounded container whose background is a full-bleed photographic or screenshot image covering the entire card area edge-to-edge. Layer a semi-transparent dimming gradient over the image so all foreground content remains readable. Anchor all foreground content to the lower-left area of the card with comfortable inset padding.

Inside each card overlay, stack from bottom to top (visually anchored to the card's lower-left corner): a short description paragraph on its own line; a prominent project title on its own line directly above the description; a horizontal row of small tech-stack or tool icons sitting side by side on the same baseline directly above the title.

On narrow viewports, keep the same stacked full-width card layout; reduce card height proportionally and ensure the overlaid text and icons remain legible with sufficient contrast against the dimmed background.
