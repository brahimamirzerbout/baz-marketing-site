# Centered Portfolio Card Grid

Create a portfolio section inside a centered page container with comfortable side padding. At the top, place a centered heading on its own line; directly below the heading, a centered supporting paragraph at a comfortable reading width.

Below the header block, place a three-column by two-row grid of six equal-width cards with consistent gutters between columns and rows. Inside each card, stack from top to bottom: a brand logo or mark centered on its own line; a project title centered on its own line directly below the logo; a short centered description paragraph directly below the title; and a text-style link at the bottom of the card composed of a short label paired with a trailing arrow icon inside a compact outlined container.

On narrow viewports, collapse the grid stepwise from three columns to two, then to a single column on the narrowest breakpoints; preserve left-to-right top-to-bottom reading order throughout.
