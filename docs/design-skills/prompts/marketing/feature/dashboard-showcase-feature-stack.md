# Dashboard Showcase with Feature Stack

Create a feature section inside a centered page container with comfortable side padding. At the top, place a left-aligned header block composed of a small pill-shaped badge (containing a leading dot marker on the left and a short label text to its right, both inside a rounded container) followed by a large multi-line heading on its own line directly below the badge.

Below the header, split the section into two columns on large 1280px width breakpoints.

The **left column** contains one tall rounded container that reads as a stage or showcase surface. Inside the stage, place a smaller rectangular panel offset as a floating card positioned slightly off-center. The floating card contains a simplified dashboard layout stacked from top to bottom: a title line; a primary numeric readout with a secondary supporting line directly beneath it; a horizontal row of compact filter chips sitting side by side; a short list of data rows where each row is a horizontal composition of a leading mark (dot or colored indicator), a label, a tiny inline sparkline graphic, a numeric value, and a small delta badge (showing a directional change indicator) anchored to the right end of the row.

The **right column** stacks three feature blocks vertically with consistent spacing between them. Each block is a horizontal row composed of a small icon anchored to the left (holding a single centered glyph) and a text column to the right of the icon containing a medium heading on its own line and a short paragraph directly beneath the heading. Separate consecutive blocks with thin horizontal rules; optionally emphasize one segment of the first divider with a thicker band so the first break reads as a progress or selection indicator.

On narrow viewports, stack with the stage column first and the feature stack second; keep all internal stacks in their original reading order.
