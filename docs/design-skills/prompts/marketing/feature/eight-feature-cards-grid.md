# Eight Feature Cards Grid

Create a feature section inside a centered page container with comfortable side padding. At the top, place a centered header block composed of a large heading centered on its own line followed by a centered supporting paragraph directly beneath it (constrain the paragraph to a moderate reading width).

Below the header, place a responsive grid of eight feature cards arranged as four columns and two rows on the widest breakpoint, with consistent gutters between all cards. Each card is its own rounded rectangular panel with a thin outer border. Inside each card, stack from top to bottom: a small rounded-square icon container at the top (holding a single centered glyph); a bold short title on its own line directly below the icon; a left-aligned supporting paragraph directly below the title that fills the remaining card width with comfortable internal padding on all sides.

On smaller breakpoints, reduce the column count stepwise (four columns, then two, then one) until the cards form a single column on the narrowest viewports; preserve left-to-right top-to-bottom reading order throughout.
