# Three Feature Showcase Cards

Create a feature section inside a centered page container with comfortable side padding. At the top, place a left-aligned header block composed of a very large multi-line heading on its own line followed immediately by a supporting paragraph directly beneath the heading at a comfortable reading width.

Below the header, place three equal-width columns on large 1280px width breakpoints that read as tall cards with consistent gutters between them. Each column is a vertical stack containing, in order: a medium heading on its own line; a short multi-line description paragraph directly below the heading; a large rectangular showcase region that fills the remaining card height below the text.

Allow each showcase region to use a different structural recipe:

**First column showcase** — layer a full-bleed photograph with a smaller centered panel floating on top of the image; the panel lists several vertical metric rows where each row is a horizontal composition of a leading symbol, a large numeric value, a two-line label, and a small corner affordance (directional arrow or action icon) anchored to the right.

**Second column showcase** — use a patterned or textured background with a loose cluster of floating pill-shaped labels scattered across the area, and a smaller anchored card overlapping the cluster from one edge so it reads as a layered composition.

**Third column showcase** — combine a speech-bubble style prompt at the top of the region with a larger card beneath it that stacks a header row, a bold title line, several horizontal skeleton lines (representing placeholder text), and a closing caption line at the bottom.

On narrow viewports, stack the three columns vertically in left-to-right reading order; preserve heading, description, then showcase order inside each column.
