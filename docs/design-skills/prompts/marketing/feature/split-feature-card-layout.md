# Split Feature Card Layout

Create a feature section framed as one large rounded outer container with comfortable outer margin on all sides so the entire section reads as a single card floating within the page.

Inside the container, use a three-column split on large 1280px width breakpoints.

The **left column** is wider than the other two and carries a left-aligned vertical stack containing, in order: a large left-aligned heading on its own line; a supporting paragraph directly beneath the heading at a comfortable reading width; a text-style link on its own line below the paragraph whose label includes a trailing arrow icon at the end of the link text.

The **middle column** and the **right column** are narrower and equal in width; each stacks two feature blocks vertically with consistent spacing between them (four blocks total across both columns). Each feature block is a left-aligned vertical stack containing, in order: a small rounded-square icon container (holding a single centered glyph); a short bold sub-heading on its own line directly below the icon; a supporting paragraph directly below the sub-heading that aligns flush-left with the icon and heading above it.

Allow a very subtle geometric grid texture to appear only behind part of the card interior (for example occupying one corner or a diagonal band) so it reads as a soft decorative surface without competing with the text content.

On narrow viewports, stack the three columns in reading order with the intro column first, the middle feature column second, and the right feature column third; keep all internal stacks in their original order.
