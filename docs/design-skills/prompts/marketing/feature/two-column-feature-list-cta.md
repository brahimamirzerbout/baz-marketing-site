# Two-Column Feature List with CTA

Create a feature section inside a centered page container with comfortable side padding. At the top, place a centered header block composed of a large heading centered on its own line followed by a centered supporting paragraph directly beneath it at a moderate reading width.

Below the header, place a two-column layout on large 1280px width breakpoints separated by a single thin vertical divider running the full height of the column area. Each column stacks three feature items vertically (six items total across both columns) with consistent spacing between them. Each item is a horizontal row composed of a small circular icon container anchored to the left (holding a single centered glyph) and a text block to the right of the icon containing a medium-weight heading on its own line and a short multi-line paragraph directly beneath the heading.

Below the two-column block, place one centered pill-shaped primary button on its own line whose label includes a trailing arrow icon at the end of the button text.

On narrow viewports, stack the two columns vertically in reading order and replace the vertical divider with a thin horizontal rule between the two halves; keep the primary button centered below both column stacks.
