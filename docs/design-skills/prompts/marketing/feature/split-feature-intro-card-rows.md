# Split Feature Intro with Card Rows

Create a feature section split into two columns on large 1280px width breakpoints inside a centered page container with comfortable side padding.

The **left column** is a left-aligned vertical stack containing, in order: a large left-aligned heading on its own line; a supporting paragraph directly beneath the heading at a comfortable reading width; a text-style link on its own line below the paragraph whose label includes a trailing arrow icon at the end of the link text.

The **right column** contains a vertical stack of three identical card rows separated by consistent vertical spacing. Each card is a horizontal row composed of three inline elements: a small rounded-square icon container anchored to the left (holding a single centered glyph); a middle text block to the right of the icon containing a bold title on its own line and a short description directly beneath the title; a compact arrow control (a small forward-pointing chevron or arrow icon) anchored to the far right of the row on the same vertical center as the icon. Cards may carry subtle boundaries (thin borders or faint background fills) to distinguish each row.

On narrow viewports, stack with the intro column first and the card stack second; keep all internal stacks in their original reading order.
