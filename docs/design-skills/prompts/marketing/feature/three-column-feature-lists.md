# Three-Column Feature Lists

Create a feature section inside a centered page container with comfortable side padding. At the top, place a left-aligned header block composed of a large heading on its own line followed by a supporting paragraph directly beneath the heading at a comfortable reading width.

Below the header, place three equal-width columns on large 1280px width breakpoints, separated by thin vertical dividers running the full height of the column area. Each column is a left-aligned vertical stack containing, in order: a small rounded-square icon container at the top (holding a single centered glyph); a medium-weight category heading on its own line directly below the icon; a vertical list of approximately five items directly below the heading. Each list item is a horizontal row composed of a small circular marker containing a checkmark icon on the left followed by a single line of text to the right on the same baseline. Keep all column content left-aligned with consistent internal padding.

On narrow viewports, stack the three columns vertically in reading order and replace the vertical dividers with thin horizontal rules between each column section.
