# Alternating Split Features with Diamond Media

Create a feature section composed of two stacked bands separated by generous vertical space inside a centered page container with comfortable side padding. Each band is a two-column split on large 1280px width breakpoints.

**First band** — the **left column** carries the text stack and the **right column** carries the media block.

**Second band** — swap so the **left column** carries the media block and the **right column** carries the text stack.

In each **text column**, stack a left-aligned vertical composition containing, in order: a large left-aligned heading on its own line; a supporting paragraph directly beneath the heading at a comfortable reading width; a thin horizontal rule spanning the column width; a vertical list of items where each item is a horizontal row composed of a small checkmark icon anchored to the left and a single line of descriptive text to the right on the same baseline; optionally a second thin horizontal rule below the list followed by a short closing paragraph on its own line.

In each **media column**, place one photograph inside a square frame with rounded corners that is rotated approximately 45 degrees so the photograph reads as a diamond shape. Behind the diamond frame, position a faint dotted diamond or lattice motif that extends slightly past the photo frame on all sides, creating a soft decorative halo around the image.

On narrow viewports, stack each band as a single column with the media block first and the text stack second; preserve all internal reading order within each stack.
