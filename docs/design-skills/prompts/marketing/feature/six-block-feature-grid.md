# Six-Block Feature Grid

Create a feature section inside a centered page container with comfortable side padding. At the top, place a left-aligned header block composed of a large multi-line heading on its own line followed by a short supporting paragraph directly beneath the heading (constrain the paragraph to a moderate reading width so it does not span the full container).

Below the header, place a three-column by two-row grid of six feature blocks with consistent gutters between columns and rows. Each block is a left-aligned vertical stack containing, in order: a small rounded-square icon container at the top (the container holds a single glyph centered inside it); a medium-weight title on its own line directly below the icon container; a short multi-line description paragraph directly below the title, left-aligned and matching the block width.

On narrow viewports collapse the grid to two columns first, then to a single column on the narrowest breakpoints, preserving left-to-right top-to-bottom reading order throughout.
