# Split Feature Section with Illustration

Create a feature section split into two columns on large 1280px width breakpoints inside a centered page container with comfortable side padding.

The **left column** is dominated by one large illustrative scene that spans most of the column height, vertically centered within the column. Allow simple supporting shapes (geometric accents or soft background forms) behind the scene if needed to fill the column area without competing with the primary illustration.

The **right column** is a left-aligned vertical stack containing, in order: a large left-aligned heading on its own line; a supporting paragraph directly beneath the heading at a comfortable reading width; a vertical list of three feature entries framed by thin horizontal rules (one rule sits above the first entry and one sits below the last entry, creating a ruled band). Each entry is a horizontal row composed of a small rounded-square icon container on the left and a text block to the right of the icon containing a medium heading on its own line, a short paragraph directly beneath the heading, and a text-style link on its own line below the paragraph whose label includes a trailing arrow icon. Below the bottom rule, finish the column with one closing paragraph that spans the full text column width.

On narrow viewports, stack with the illustration column first and the text column second; keep all internal stacks in their original reading order.
