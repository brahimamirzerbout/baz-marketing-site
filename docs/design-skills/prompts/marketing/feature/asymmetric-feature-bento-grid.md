# Asymmetric Feature Bento Grid

Create a feature section inside a centered page container with comfortable side padding. At the top, place a centered header block composed of a large heading centered on its own line followed by a centered supporting paragraph directly beneath it at a comfortable reading width.

Below the header, place a two-row asymmetric grid of four feature cards with consistent gutters between all cards. The first row splits into one wide card on the left (occupying roughly two-thirds of the row width) and one narrow card on the right (occupying the remaining third); the second row reverses so the narrow card sits on the left and the wide card sits on the right, creating a staggered bento composition.

Each card is a rounded rectangular panel containing a left-aligned vertical stack, in order: a left-aligned title on its own line; a short description paragraph directly below the title; a large simplified UI mockup region that fills the remaining card area — the mockup may be cropped by the card edges on the bottom and sides so it reads as a window into a larger interface.

On narrow viewports, stack the four cards in a single column in reading order (top-left, top-right, bottom-left, bottom-right) while preserving title, description, then mockup order inside each card.
