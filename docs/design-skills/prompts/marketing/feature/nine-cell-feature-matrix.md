# Nine-Cell Feature Matrix

Create a feature section inside a centered page container with comfortable side padding. At the top, place a centered header block composed of a large heading centered on its own line followed by a centered supporting paragraph directly beneath it (constrain the paragraph to a moderate max line length so it does not span the full container width).

Below the header, place one large inner container with rounded corners and a thin outer boundary that frames the entire grid area. Inside that container, build a three-column by three-row grid of nine feature cells with no gutters; instead, separate adjacent cells with very thin rules so columns and rows read as a matrix. Inset the rules slightly from the inner container edges rather than running flush to the corners, leaving small rounded pockets at each corner of the container.

Each cell is a centered vertical stack containing, in order: a small circular icon container at the top (holding a single centered glyph); a bold title on its own line directly below the icon; a short centered description paragraph directly below the title. Keep all content within each cell horizontally centered.

On narrow viewports, collapse the grid into a single column and replace vertical rules with horizontal ones between rows so each cell remains visually separated; preserve top-to-bottom reading order.
