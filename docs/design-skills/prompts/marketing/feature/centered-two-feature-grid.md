# Centered Two-Feature Grid

Create a feature section inside a centered page container with comfortable side padding where every primary text block is horizontally centered. At the top, place a large heading centered on its own line followed by a centered supporting paragraph directly beneath it (constrain the paragraph to a restrained line width so it reads as a compact centered block).

Below the header, place a two-column grid on large 1280px width breakpoints with one feature in each column and consistent gutters between them. Each feature is a centered vertical stack containing, in order: a small rounded-square icon container centered above the text (holding a single centered glyph); a bold heading centered on its own line directly below the icon; a centered description paragraph directly below the heading, constrained to the column width.

Beneath the two-column grid, add a single centered text link on its own line whose label includes a trailing arrow icon at the end of the link text.

On narrow viewports, stack the two features vertically in reading order with the same centered alignment inside each feature block; keep the footer link centered below both.
