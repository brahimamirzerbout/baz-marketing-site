# Split Media Band with Numbered Features

Create a feature section split into two equal-width columns on large 1280px width breakpoints that spans the full available width of the page container.

The **left column** is a full-height media band: a single photographic scene anchored edge-to-edge within the column, filling the entire column area. Position a very subtle dotted grid motif tucked into one corner of the media band as a light structural accent that does not compete with the photograph. Over the lower third of the media band, overlay a centered content block containing a short multi-line paragraph on its own line with one pill-shaped primary button placed directly beneath the paragraph, also centered horizontally.

The **right column** is a vertical stack containing, in order: a very large multi-line heading at the top of the column; a two-by-two grid of four feature blocks with consistent gutters positioned below the heading. Each block is a vertical stack composed of a large numeric label (reading as "01", "02", and so on) on its own line followed by a short multi-line paragraph directly beneath the number. Allow a single oversized letterform watermark to sit behind the entire right column as a decorative background element, spanning across multiple blocks without interrupting the text stack above it.

On narrow viewports, stack with the media band (including its overlay content) first, then the heading and the four numbered blocks in a single column beneath; preserve all internal reading order.
