# Centered Dual-Button CTA

Create a CTA section as one centered vertical stack inside a centered page container with comfortable side padding. At the top, place a large multi-line primary heading centered horizontally on its own line. Below the heading, place a supporting paragraph set at a restrained max width and centered horizontally so long lines do not stretch edge to edge.

Below the paragraph, center a single horizontal row containing two buttons sitting side by side with a small gap between them (approximately 16 px). The first button is a solid primary brand button with a short label; the second button is a secondary outlined button with a short label. Both buttons use pill-shaped or softly rounded corners and share the same height so they read as one balanced pair. The row itself is centered beneath the paragraph (one flex row, items centered, never wrapping to a second line on any viewport wider than the minimum supported width).
