# Two-Column Diamond Image CTA

Create a CTA section split into two columns on large 1280 px width breakpoints inside a centered page container with comfortable side padding.

The **left column** is a left-aligned vertical stack containing, in order from top to bottom: a large two-line heading on its own line; a first body paragraph at a comfortable reading width directly beneath the heading; a second body paragraph directly beneath the first; a single horizontal row containing a pill-shaped primary button with a small trailing rightward arrow at the right of its label and a plain text link with a small trailing rightward arrow placed to the right of the button **on the same horizontal row** (one flex row, items vertically centered on the same baseline, never stacked — the button sits at the leading edge of the row and the text link follows after a small gap).

The **right column** shows one photograph inside a square frame that is rotated forty-five degrees so the media reads as a diamond shape. Keep the frame's own corners softly rounded so the diamond silhouette stays legible and does not appear as a sharp rhombus. Vertically center the diamond frame relative to the text column when the two columns differ in height. Clip the photograph to the rotated frame so no overflow is visible.

On narrow viewports, stack with the text column first and the diamond-framed image beneath it, centered horizontally.
