# Two-Column QR CTA

Create a CTA section split into two columns on large 1280 px width breakpoints inside a centered page container with comfortable side padding.

The **left column** is a left-aligned vertical stack containing, in order from top to bottom: a large heading on its own line; a short sub-headline line directly beneath the heading on its own line; a multi-line body paragraph at a comfortable reading width directly beneath the sub-headline; a text link whose label includes a small trailing rightward arrow affordance at the end of the link text (the link sits on its own line beneath the paragraph).

The **right column** centers one square QR code graphic both horizontally and vertically within the column. The QR code is framed by a light "corner bracket" treatment — four small right-angle bracket shapes positioned at the four corners of the code so it reads as a scanning target. Vertically center the entire QR block relative to the text column when the two columns differ in height.

On narrow viewports, stack with the text column first and the QR code block beneath it, centered horizontally.
