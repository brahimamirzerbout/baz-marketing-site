# Two-Column Diamond Media Content Section

Create a content section split into two columns on large 1280px width breakpoints inside a centered page container with comfortable side padding. The two columns share the same vertical center line.

The **left column** is a left-aligned vertical stack containing, in order: a large heading on its own line; an introductory paragraph directly beneath the heading at a comfortable reading width; then a vertical list of three feature items with consistent spacing between them — each item is a horizontal row pairing a small square icon container at the left with a single-line label to the right of the icon on the same baseline. Below the list, add one closing paragraph that continues the narrative, then one primary CTA button whose label pairs with a small trailing arrow icon at the right end of the label.

The **right column** contains one photograph placed inside a square frame with generously rounded corners. The frame is rotated approximately forty-five degrees so the photo reads as a diamond shape relative to the section grid. Vertically center the diamond frame relative to the left column so both columns share the same visual midpoint. The photograph fills the rotated frame edge to edge with overflow clipped.

On narrow viewports, stack the text-and-button column above the media column; the diamond frame may scale down proportionally to fit the single-column width.
