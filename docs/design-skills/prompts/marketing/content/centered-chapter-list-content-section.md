# Centered Chapter List Content Section

Create a content section as a vertical stack inside a centered page container with comfortable side padding.

The **heading** sits at the top and is horizontally centered on its own line.

The **chapter container** sits below the heading. Place one large rounded rectangular container that spans most of the content width (roughly three-quarters to full width) and is horizontally centered. Inside the container, stack six chapter entries vertically with consistent spacing between them; each entry is a vertical pair containing a bold chapter title line on top and a multi-line descriptive paragraph directly beneath the title. Separate each entry from the next with enough vertical space to read as distinct sections. The container carries a visible border or subtle filled surface so it reads as a single cohesive card.

The **footer link** sits below the chapter container. Center a single text-style link whose label includes a small trailing arrow icon, placed on its own centered line with comfortable spacing above it.

On narrow viewports, keep the chapter container at full content width with comfortable horizontal padding; preserve all six entries in their original stacking order.
