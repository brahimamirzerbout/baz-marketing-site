# Centered Header With Staggered Image Grid

Create a content section as a vertical stack inside a centered page container with comfortable side padding.

The **header block** sits at the top and is horizontally centered: a large heading on its own line, then a centered supporting paragraph directly beneath the heading at a comfortable reading width.

The **CTA row** sits directly below the header block. Center two CTA buttons in a horizontal row (a primary button whose label includes a small trailing arrow icon at the right end, and a secondary button whose label pairs with a small leading eye icon at the left end). Keep consistent spacing between the two buttons.

The **image grid** sits below the CTA row. Lay out a two-row arrangement on a four-column underlying grid with consistent gutters. The top row contains, from left to right: one wide tile spanning the first two columns, a square-ish tile occupying the third column, and a square-ish tile occupying the fourth column. The bottom row contains, from left to right: a square-ish tile occupying the first column, one wide tile spanning the second and third columns, and a square-ish tile occupying the fourth column — so that wide and narrow tiles alternate for a staggered composition. Every cell holds a single image or placeholder region with rounded corners.

On narrow viewports, reflow the grid into a two-column or single-column stack while keeping top-to-bottom, left-to-right reading order sensible; allow tiles to fill the available width with comfortable horizontal padding.
