# Split Intro With Stat Row

Create a content section as a vertical stack inside a centered page container with comfortable side padding.

The **top band** is a two-column split on large 1280px width breakpoints. The two columns share the same vertical center line.

The **left column** of the top band is a left-aligned vertical stack containing, in order: a large heading on its own line; a supporting paragraph directly beneath the heading at a comfortable reading width; and one CTA button whose label pairs with a small trailing arrow icon at the right end of the label. Keep consistent vertical spacing between each element.

The **right column** of the top band is a left-aligned vertical stack containing three short paragraphs arranged one after another with consistent spacing between them; the paragraphs together form a continuous narrative block.

The **stat row** sits below the two-column band as a full-width horizontal row spanning the entire content area. Place four equal-width stat blocks side by side with consistent gutters. Each block is a compact vertical stack showing one prominent numeric or shorthand value (set at a large display size) on top and a short descriptive label directly beneath it.

On narrow viewports, stack the left column above the right column, then place the stat row beneath both; allow the four stat blocks to wrap into a two-by-two grid or scroll horizontally as needed.
