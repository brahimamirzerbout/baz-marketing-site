# FAQ Support Blocks Panel

Create a FAQ section inside a centered page container with comfortable side padding. At the top, place a large centered heading on its own line.

Below the heading, place one large rounded container that spans most of the container width. Inside that container, arrange three equal-width support blocks in a single horizontal row on large breakpoints with consistent gutters between them. Each block is a left-aligned horizontal composition of a small icon anchored to the left and a text group to the right of the icon containing a bold title on its own line, a short multi-line description paragraph directly below the title, and a text-style link on its own line at the bottom of the block whose label includes a trailing arrow icon.

On narrow viewports stack the three support blocks vertically in reading order within the rounded container; each block stretches to the full available width with comfortable internal padding.
