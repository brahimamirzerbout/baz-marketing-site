# Searchable FAQ Topic Grid

Create a FAQ section inside a centered page container with comfortable side padding. At the top, place a centered header block composed of a large centered heading on its own line followed by a centered supporting paragraph directly beneath it at a moderate reading width.

Below the header, place a single centered search input spanning a comfortable width on its own line. The input contains a small leading search icon inside the field and placeholder text.

Below the search input, place a three-column by two-row grid of six topic cards with consistent gutters between columns and rows. Each card is a rounded rectangular panel containing a left-aligned vertical stack, in order: a small rounded-square icon container at the top (holding a single centered glyph); a bold category title on its own line directly below the icon; a short vertical list of question links (each link on its own line as plain text); and a text-style link at the bottom of the card whose label reads as a "show all" action.

On narrow viewports collapse the grid to two columns first, then to a single column on the narrowest breakpoints; preserve left-to-right top-to-bottom reading order throughout.
