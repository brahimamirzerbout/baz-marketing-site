# FAQ Header Search With Link Columns

Create a FAQ section inside a centered page container with comfortable side padding. At the top, place a horizontal header row on large breakpoints containing a left-aligned block on the left and a search input anchored to the right on the same baseline. The left block is a vertical stack composed of a large left-aligned heading on its own line and a supporting paragraph directly beneath the heading at a comfortable reading width. The search input contains a small leading search icon inside the field and placeholder text.

Below the header, place a thin horizontal rule spanning the full container width.

Below the rule, place two equal-width columns on large breakpoints with consistent gutters between them. Each column is a left-aligned vertical stack containing a bold category heading on its own line at the top followed by a vertical list of question links, one per line, directly beneath the heading. Each link reads as plain text on its own line.

Below the two-column block, place a full-width footer bar separated by a thin horizontal rule. The bar is a horizontal row containing a short support prompt on the left and a compact button on the right on the same baseline, with the button label preceded by a small leading icon.

On narrow viewports stack the heading block above the search input, stack the two link columns vertically in reading order, and center the footer bar elements vertically.
