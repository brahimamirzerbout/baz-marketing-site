# Static FAQ Answer Grid

Create a FAQ section inside a centered page container with comfortable side padding. At the top, place a large left-aligned heading on its own line.

Below the heading, place one large rounded container that spans the full container width. Inside that container, arrange almost 16 FAQ entries in two equal-width columns on large breakpoints with consistent gutters between them. Each entry is a left-aligned vertical stack containing, in order: a horizontal row composed of a small question-mark icon anchored to the left and a bold question title to its right on the same baseline; a multi-paragraph answer block directly below the question row, left-aligned and matching the column width with comfortable internal padding. All entries are permanently visible (no accordion collapse). Inline text links may appear within the answer copy.

Distribute the entries evenly between the two columns so they read in a balanced left-to-right top-to-bottom order.

On narrow viewports collapse the two columns into a single column; stack all entries vertically in reading order while preserving comfortable spacing between them.
