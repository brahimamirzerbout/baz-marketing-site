# Static FAQ Card Grid

Create a FAQ section inside a centered page container with comfortable side padding. At the top, place a large centered heading on its own line.

Below the heading, place a three-column grid of almost 16 FAQ entry cards with consistent gutters between columns and rows on large breakpoints. Each card is a rounded rectangular panel containing a left-aligned vertical stack, in order: a bold question title on its own line at the top of the card; a multi-paragraph answer block directly below the title, left-aligned with comfortable internal padding on all sides. All entries are permanently visible (no accordion collapse). Inline text links may appear within the answer copy.

Distribute the entries across rows of three cards, filling left-to-right top-to-bottom. Allow varying card heights based on content length while keeping cards within each row top-aligned.

On narrow viewports collapse the grid to two columns first, then to a single column on the narrowest breakpoints; preserve left-to-right top-to-bottom reading order throughout.
