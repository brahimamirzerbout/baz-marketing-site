# Team Narrative With Profile Grid

Create a team section split into two columns on large 1280px width breakpoints inside a centered page container with comfortable side padding.

The **left column** is a left-aligned vertical stack containing, in order: a large left-aligned multi-line heading on its own line; a supporting paragraph directly beneath the heading at a comfortable reading width; a second paragraph directly below the first that elaborates on the team narrative at the same reading width. Below the paragraphs, after extra vertical space, place a horizontal row of three stat blocks on the same baseline; each block is a compact vertical stack composed of one large prominent numeric value (e.g. "10k+") on its own line at the top and a short descriptive label directly beneath the value. Below the stat row, place a single compact outlined button on its own line whose label is composed of a small leading icon and a short action label on the same baseline.

The **right column** contains a two-column by two-row grid of four team member blocks with consistent gutters between them. Each block is a centered vertical stack containing, in order: a medium circular avatar image at the top; the team member's name on its own line directly below the avatar in medium weight; a single-line job title directly below the name in lighter weight; and a horizontal row of small social-media icon links sitting side by side on the same baseline directly below the title.

On narrow viewports stack the two columns vertically with the description column first and the team grid directly beneath it; preserve all internal stacks in their original reading order.
