# Three-Day Event Schedule Panels

Create an event schedule section as one vertically stacked block on a full-width background inside a max-width container. At the top, place a centered vertical stack containing: a small inline element composed of a tiny clock icon and a short timezone label on the same baseline, centered on its own line; a large centered heading on its own line directly below the timezone element; a centered date or date-range line directly below the heading.

Below the header block, place three equal-width day columns in one horizontal row on large 1280px width breakpoints with consistent gutters between them. Each column is a rounded panel with a day title (e.g. "Wednesday 27th") anchored at the top of the panel on its own line.

Below the day title, stack a vertical list of 16 session blocks separated by comfortable vertical spacing and an horizontal divider. Each session block is a left-aligned vertical stack containing, in order: a small inline time-range element composed of a tiny clock icon and a short start–end time label on the same baseline, on its own line; a medium-weight session title on its own line directly below the time range — every session title must be a link; an optional short description or talk title directly below the session title.

Session blocks that represent talks include a speaker row directly below the title: a horizontal element composed of a small circular avatar on the left, and a two-line text block to its right on the same baseline containing the speaker name on the first line and the speaker role or affiliation on the second line. When a session has multiple speakers, stack additional speaker rows vertically beneath the first, each following the same avatar-plus-text pattern.

Session blocks that represent breaks or social events (e.g. coffee, networking, after-party) replace the speaker row with a sponsor row: a short "Sponsors:" label on its own line followed by a horizontal row of small brand logos sitting side by side on the same baseline directly below the label.

On narrow viewports, stack the three day columns vertically in their original left-to-right reading order; each column stretches to the full content width with comfortable side padding.
