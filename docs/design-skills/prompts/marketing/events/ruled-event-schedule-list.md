# Ruled Event Schedule List

Create an event schedule section as one vertically stacked block inside a max-width container. At the top, place a large centered heading on its own line; directly below the heading, a centered text link whose label includes a trailing arrow icon at the end of the link text, on its own line.

Below the header, place a vertical list of session rows separated by thin horizontal rules spanning the full container width (one rule sits above the first row and one sits below the last row, creating a ruled band). Each session row is a single horizontal band composed of two inline elements on the same baseline: a time-range label left-aligned within a fixed-width leading column (e.g. "08:00 - 09:00"), and a session title to the right of the time column on the same baseline — every session title must be a link. Keep the time column width consistent across all rows so the session titles align vertically.

On narrow viewports, keep the same two-column row structure; if the container becomes too narrow for side-by-side alignment, stack the time label above the session title within each row, preserving the thin horizontal rules between rows.
