# Centered Login Card Page

Create a login page as one centered vertical stack on a full-page background. At the very top, place a small brand mark composed of a circular logo glyph followed by a short brand wordmark, both centered horizontally on the page. Below the brand mark, place one centered form card with rounded corners and a moderate max width. Inside the card, stack from top to bottom: a card heading on its own line; a labeled email input row (the label sits above the input on its own line and pairs a short label text with a tiny required-field marker to the right of the label; the input includes a small leading icon glyph inside the field); a labeled password input row in the same shape directly below (also carrying a small leading icon glyph inside the field); a single horizontal row composed of a small square "remember me" checkbox paired with an inline label on the left and a "forgot password" text link on the right; one full-width primary submit button beneath the checkbox row; finally, a small centered helper line containing a short prompt sentence followed by an inline text link to the sign-up flow. Treat the area outside the card as a subtle decorative field (for example a sparse dot or grid pattern). On narrow viewports keep the same single-column stack inside the card and let the card occupy the full content width with comfortable side padding.
