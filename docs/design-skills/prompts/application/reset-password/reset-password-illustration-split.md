# Reset Password With Illustration Split

Create a reset-password page split into two columns on large 1280px width breakpoints inside a centered page container with comfortable side padding.

**Left column** contains a single elevated card with rounded corners and a light surface treatment, vertically centered within the column height. Inside the card, stack from top to bottom: a small brand icon or logo mark on its own line paired with a brand name label immediately to the right of the icon on the same baseline; a bold heading on its own line directly below the brand row (e.g. "Change you password."); a short supporting sentence directly below the heading explaining that the new password must differ from previous ones; a labeled email input row placed below the supporting text (the label sits above the input on its own line with a tiny required-field marker to the right of the label text; the input carries a leading envelope icon affordance flush-left inside the field and a placeholder suggesting an email address); a two-column field row placed below the email input with comfortable column gaps — required password input on the left (label above with required-field marker; masked dots placeholder inside) and required repeat-password input on the right (same shape); a single checkbox row spanning the full form width containing a small square checkbox at the left and an inline label whose text confirms acceptance of linked terms and conditions (with the terms phrase rendered as an inline text link); one full-width primary action button at the bottom of the card with a short label (e.g. "Reset password").

**Right column** holds one large illustrative scene that fills most of the column area, vertically centered within the column. The illustration depicts a figure sitting at a workstation with a monitor displaying a user profile or authentication interface, plants or office elements in the background, conveying a calm account-recovery theme without relying on photographic imagery.

On narrow viewports stack the two columns vertically with the form card first and the illustration directly below it; keep all internal stacks in their original reading order with comfortable side padding.
