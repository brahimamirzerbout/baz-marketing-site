# Full-Width Split Reset Password Page

Create a reset-password page split into two columns on large 1280px width breakpoints that spans the full available width of the viewport with no outer margin.

**Left column** uses a plain page background and contains a left-aligned vertical stack constrained to a 424px max width, vertically and horizontally centered within the column. Stack from top to bottom: a bold heading on its own line (e.g. "Create a new password."); a short supporting sentence directly below the heading explaining that the new password must differ from previous ones; a two-column field grid placed below the supporting text after extra vertical space with comfortable column gaps — required name input on the left (label above with required-field marker; first-name placeholder inside) and required email input on the right (label above with required-field marker; email placeholder with leading envelope icon affordance inside); required password input on the left (label above with required-field marker; masked dots placeholder inside) and required repeat-password input on the right (same shape); a single checkbox row spanning the full form width containing a small square checkbox at the left and an inline label whose text confirms acceptance of linked terms and conditions (with the terms phrase rendered as an inline text link); one full-width primary action button at the bottom with a short label (e.g. "Reset password").

**Right column** uses a distinct filled background that spans the full column area edge-to-edge. Inside the column, place a centered vertical stack vertically centered within the column height containing: a small brand icon or logo mark on its own line paired with a brand name label immediately to the right of the icon on the same baseline; a large bold heading on its own line directly below the brand row delivering a compelling value statement (e.g. "Explore the world's leading design portfolios."); a supporting paragraph of one or two sentences directly below the heading at a comfortable reading width describing the platform community; a social proof row placed below the paragraph, composed of slightly overlapping circular user avatars on the left and a short line of text immediately to the right of the avatars on the same baseline stating a review metric (e.g. "Rated best over 15.7k reviews").

On narrow viewports stack the two columns vertically with the form column first and the branded column directly below it; each internal stack preserves its original reading order with comfortable side padding.
