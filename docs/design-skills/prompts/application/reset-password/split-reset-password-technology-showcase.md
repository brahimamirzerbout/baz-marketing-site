# Split Reset Password With Technology Showcase

Create a reset-password page split into two columns on large 1280px width breakpoints inside a centered page container with comfortable side padding.

**Left column** is a left-aligned vertical stack, vertically centered within the column height, containing a compact reset-password form. Stack from top to bottom: a bold heading on its own line (e.g. "Create new password"); a labeled email input row placed below the heading after extra vertical space (the label sits above the input on its own line with a tiny required-field marker to the right of the label text; the input carries a leading envelope icon affordance flush-left inside the field and a placeholder suggesting an email address); a two-column field row placed below the email input with comfortable column gaps — required new-password input on the left (label above with required-field marker; masked dots placeholder inside) and required confirm-password input on the right (same shape); a single checkbox row spanning the full form width containing a small square checkbox at the left and an inline label whose text confirms acceptance of linked terms and conditions (with the terms phrase rendered as an inline text link); one full-width primary action button at the bottom with a short label (e.g. "Reset password").

**Right column** uses a subtle filled background with rounded corners and contains a centered vertical stack vertically centered within the column height. Inside the column, place a circular arrangement of technology or partner brand icons distributed evenly around an invisible orbital ring (each icon sits inside a small rounded container); below the orbital graphic, place a bold centered heading stating a technology count or integration metric on its own line (e.g. "Over 200+ technologies"); a centered supporting paragraph of one or two sentences directly below the heading describing the platform's integration breadth.

On narrow viewports stack the two columns vertically with the form column first and the technology showcase directly below it; keep all internal stacks in their original reading order with comfortable side padding.
