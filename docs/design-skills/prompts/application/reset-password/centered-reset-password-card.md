# Centered Reset Password Card

Create a reset-password page as one centered vertical stack on a full-width section background that may include a very subtle repeating dot or grid texture behind the content, vertically centered within the viewport height. Stack from top to bottom: a small brand icon or logo mark centered horizontally on its own line paired with a brand name label immediately to the right of the icon on the same baseline; a single elevated card with rounded corners placed directly below the brand row, centered horizontally and constrained to a comfortable form width. Inside the card, stack from top to bottom: a bold heading on its own line (e.g. "Change password"); a labeled email input row placed below the heading (the label sits above the input on its own line with a tiny required-field marker to the right of the label text; the input carries a leading envelope icon affordance flush-left inside the field and a placeholder suggesting an email address); a labeled new-password input row directly below (label above with required-field marker; masked dots placeholder inside); a labeled confirm-password input row directly below (same shape); a single checkbox row containing a small square checkbox at the left and an inline label whose text confirms acceptance of linked terms and conditions (with the terms phrase rendered as an inline text link); one full-width primary action button at the bottom of the card with a short label (e.g. "Reset password").

On narrow viewports keep the same single-column centered stack with comfortable side padding; allow the card to span the full content width while preserving center alignment throughout.
