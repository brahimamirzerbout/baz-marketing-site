# Full-Width Split Maintenance Page With Resources

Create a maintenance page split into two columns on large 1280px width breakpoints that spans the full available width of the viewport with no outer margin.

**Left column** uses a plain page background and contains a left-aligned vertical stack with comfortable inset padding, vertically centered within the column height. Stack from top to bottom: a large multi-line heading delivering a positive forward-looking message on its own line (e.g. "We've got something special for you"); a supporting paragraph of one or two sentences directly below the heading at a comfortable reading width, explaining that the site is temporarily down for maintenance and will return with improvements; a social proof row placed below the paragraph after extra vertical space, composed of slightly overlapping circular user avatars on the left and a short line of bold text immediately to the right of the avatars on the same baseline stating a usage metric (e.g. "Used by over 15M people around the world"); a compact outlined button placed below the social proof row on its own line, composed of a small leading chat or message icon paired with a short label (e.g. "Speak with us").

**Right column** uses a distinct filled background that spans the full column area edge-to-edge, with a very subtle geometric wave or line texture layered behind the content. Inside the column, place a vertical stack of three resource cards with consistent spacing between them, vertically centered within the column height. Each card is a left-aligned vertical stack composed of a small rounded-square icon container at the top (holding a single centered glyph); a bold title on its own line directly below the icon; a short two-line description paragraph directly below the title; and a text-style link on its own line below the paragraph whose label includes a trailing arrow icon at the end of the link text.

On narrow viewports stack the two columns vertically with the left column first and the resource column directly below it; each internal stack preserves its original reading order with comfortable side padding.
