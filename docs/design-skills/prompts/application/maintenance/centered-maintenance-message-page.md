# Centered Maintenance Message Page

Create a maintenance page as one centered vertical stack on a full-width section background, vertically centered within the viewport height. Stack from top to bottom: a small brand icon or logo mark centered horizontally on its own line; a brand name label centered on its own line directly below the icon; a large bold centered heading on its own line directly below the brand name delivering a positive forward-looking message (e.g. "We've got something special for you"); a single centered paragraph of one or two sentences directly below the heading at a comfortable reading width, explaining that the site is temporarily down for maintenance and will return with improvements.

On narrow viewports keep the same single-column centered stack with comfortable side padding; allow the heading to wrap naturally while preserving center alignment throughout.
