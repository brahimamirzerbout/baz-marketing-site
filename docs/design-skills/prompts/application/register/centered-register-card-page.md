# Centered Register Card Page

Create a register page as one centered vertical 448px card on a full-page background. At the very top, place a small brand mark composed of a circular logo glyph followed by a short brand wordmark, both centered horizontally on the page. Below the brand mark, place one centered form card with rounded corners and a moderate max width. Inside the card, stack from top to bottom: a card heading on its own line; a labeled email input row (the label sits above the input on its own line and pairs a short label text with a tiny required-field marker to the right of the label); a labeled password input row in the same shape directly below; a labeled repeat-password input row in the same shape directly below that; one checkbox row containing a small square checkbox at the left and an inline label whose text includes a single in-line text link; one full-width primary button beneath the checkbox row; finally, a small centered helper line containing a short prompt sentence followed by an inline text link to the sign-in flow. Treat the area outside the card as a subtle decorative field (for example a sparse dot or grid pattern). On narrow viewports keep the same single-column stack inside the card and let the card occupy the full content width with comfortable side padding.
