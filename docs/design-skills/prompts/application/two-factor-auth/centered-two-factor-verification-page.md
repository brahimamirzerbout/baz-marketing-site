# Centered Two-Factor Verification Page

Create a two-factor authentication page as one centered vertical stack on a full-width section background, vertically centered within the viewport height. Stack from top to bottom: a small brand icon or logo mark centered horizontally on its own line; a brand name label centered on its own line directly below the icon; a large bold centered heading on its own line directly below the brand name stating the purpose of the page (e.g. "Two-factor authentication"); a centered paragraph of one or two sentences directly below the heading at a comfortable reading width, instructing the user to open their authenticator app to view the code and verify their identity; a labeled field group placed below the paragraph after extra vertical space, composed of a short left-aligned label on its own line (e.g. "Enter verification code") above a horizontal row of six equally sized square single-character numeric input fields on the same baseline (each field is a real input element that accepts exactly one digit, auto-advances focus to the next field on entry, supports backspace to move focus back to the previous field, and the currently focused field displays a visible focus ring or cursor indicator); a short helper line directly below the input row at a subdued weight explaining where the code was sent; one primary action button placed below the helper text on its own line, composed of a small leading checkmark icon paired with a short label (e.g. "Verify").

On narrow viewports keep the same single-column centered stack with comfortable side padding; allow the six input fields to scale slightly while preserving center alignment throughout.
